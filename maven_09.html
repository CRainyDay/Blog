Maven_Web工程的自动部署。

1.Maven_Web工程的手动部署

  • ①Eclipse中，在要部署的Web工程的pom.xml上，右键单击→Run as→Maven build…：
  • 在 Goals 中填写打包 package 命令，再点击 Run 运行。就会打包Web项目为war包，(需要 Refresh 一下 target 目录)如图：

  • ②将war包放进Servlet容器的webapps目录，并重命名：

  • ③启动你的Servlet容器，便会自动解压war包：

  • ④浏览器访问，http://localhost:你的Servlet容器端口号/myweb/ 即可成功访问到：

2.Maven_Web工程的自动部署

  • ①配置Web工程的pom.xml文件，在projet节点下：
  • 注意：可以在 cargo官网，查看具体的Maven配置启动Servlet容器的信息。
One More Thing 👇
  • ②Eclipse中，在要部署的Web工程的pom.xml上，右键单击→Run as→Maven build…：
  • 在 Goals 中填写打包 deploy 命令，再点击 Run 运行。即可自动部署到本地的Servlet容器。
  • 但是在Eclipse中无法停止自动部署时，启动的 Servlet 容器，需要手动停止。
